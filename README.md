🌐 Responsive Landing Page – HTML & CSS Project

📖 Project Overview
This project is a simple and fully responsive Landing Page developed as part of Task 2 – Responsive Landing Page.

The application demonstrates how to design a modern landing page for a product or service using HTML and CSS. The page is optimized for both mobile and desktop devices.

🚀 Features
Hero Section with attractive heading and call-to-action
Features Section with 3–4 feature cards
Responsive layout for mobile and desktop
Clean and modern UI design
Footer section with basic details

🛠️ Tech Stack
HTML
CSS

▶️ How It Works
User opens the landing page
Views hero banner and content sections
Layout automatically adjusts based on screen size
Smooth experience on mobile and desktop

💻 How to Run
Clone the repository
Open index.html in a browser
View on desktop or mobile screen
